> **"Fancy-looking DLC blocks should actually BE fancy."**

Space Engineers DLC comes with a lot of gorgeous reskins — Warfare 2 reactors, Sci-Fi hydrogen engines, Sparks of the Future ion thrusters, Heavy Industry cargo containers — but they're mechanically identical to their vanilla counterparts. **NJFL fixes that.** Buffed stats, tiered variants, and a Superconductor tax that scales with block size so it stays balanced.

## ✨ What it does

Blocks get one of two prefixes based on how much they were buffed:

| Prefix | Multiplier | Component cost | Superconductor tax | Reactor Component tax |
|---|---|---|---|---|
| **Upgraded** | ~2× vanilla | 2× | 10 / 5 / 4 / 2 (LG-L / LG-S / SG-L / SG-S) | — |
| **Advanced** | ~3× vanilla | 3× | Same scale | 5 / 2 / 2 / 1 (gates behind Platinum) |

All impacted blocks also get **doubled durability** to match the added components. Names remain the same as vanilla — **your DLC blocks won't vanish if you install or remove the mod.**

## 🎯 What's covered

Buffed blocks from these DLCs (see the [Steam Workshop page](https://steamcommunity.com/sharedfiles/filedetails/?id=3572467804) for the full list per pack):

- **Apex Survival Pack** — Welder/Grinder/Drill/Ore Detector Type II
- **Fieldwork Pack** — Lab H2/O2 tanks + generators
- **Contact Pack** — Gatling + Rocket Turret Type II
- **Warfare 2** — Warfare Reactor (2× power, ½ fuel), Warfare Battery, weapons, ion thrusters (→ Advanced)
- **Heavy Industry** — Industrial Refinery/Assembler, Large Hydro/Cargo reskins, Industrial Hydro thrusters (→ Upgraded)
- **Sparks of the Future** — Sci-Fi ion + atmospheric thrusters (→ Upgraded)
- **Warfare Evolution** — Flat atmospheric thrusters (→ Advanced)
- **Decorative Pack 3** — Twin Blade Wind Turbine (2× power)
- **Core Systems Pack** — Sci-Fi Hydrogen Thruster (→ Advanced), Sci-Fi Hydrogen Engine (2× fuel cap, ½ H2/MW)

Weapons get durability + cargo buffs but **no damage increase** — turrets get a small range buff only.

## 🚀 Install

1. [Subscribe on the Workshop page](https://steamcommunity.com/sharedfiles/filedetails/?id=3572467804)
2. Enable the mod when creating or loading a world
3. Build any DLC block — the upgraded/advanced variants appear on the same block's variant arrow cycle

## 🧩 Companion mods

- **[Not Just For Looks — Additions](https://github.com/gitpush-mod/se-not-just-for-looks-additions)** — adds the missing Base + Upgraded tiers for Flat Atmospheric Thrusters, so the cycle goes Base → Upgraded → Advanced
- **[Not Just For Looks — Without Weapons](https://github.com/gitpush-mod/se-not-just-for-looks-without-weapons)** — same mod, weapons removed (for servers that don't want NJFL touching combat balance)

## 🎯 Compatibility

- ✅ **Space Engineers 1** — actively maintained
- ✅ **Save-safe** — installs/removes without breaking existing worlds (this is a data mod, not a block-swap mod)
- ✅ **Multiplayer + dedicated servers**
- ⚠️ **Other block-adjustment mods** — if another mod also modifies the same DLC blocks, **mod load order matters**. NJFL keeps working on blocks other mods aren't touching.
